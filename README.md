
# BlueBerry

Проект **BlueBerry** представляет собой бота для Discord, который включает в себя различные модули для управления сервером, экономики, команд поддержки и развлечений. Он предлагает функционал для приветственных сообщений, установки ролей для новых участников, а также множество развлекательных команд для пользователей.

## Структура проекта

Проект имеет следующую структуру:

```plaintext
BlueBerry/
│   main.py          # Основной файл для запуска бота
│   README.md        # Этот файл
│
├───Aseets          # Ресурсы (изображения, шрифты и т.д.)
│   ├───Exemple     # Пример изображения
│   │       LV.png
│   │
│   ├───Font        # Шрифты
│   │       Panton-Black.ttf
│   │
│   └───Source      # Источник
├───database        # Базы данных
├───json            # JSON файлы для настроек и команд
│       commands.json
│
├───module             # Модули бота
│       Admin.py       # Модуль администратора
│       Command.py     # Команды бота
│       Economy.py     # Экономика
│       Error.py       # Обработка ошибок
│       Fan.py         # Развлекательные команды
│       Help.py        # Команды помощи
│       Information.py # Информация о сервере и пользователе
│       Support.py     # Команды поддержки
│       Welcome.py     # Приветственные сообщения
│
├───setting         # Настройки бота
│       color.py     # Настройки цветов
│       config.py    # Основные настройки
│
└───Utils           # Утилиты
        SColor.py   # Утилиты для работы с цветами
```

## Установка и запуск

### Требования
- Python 3.8+
- Библиотеки:
  - disnake
  - pillow
  - sqlite3
  - aiohttp

### Установка зависимостей
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/SWAY-gg/BlueBerry.git
   ```
2. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```

### Запуск
1. Перейдите в каталог проекта:
   ```bash
   cd BlueBerry
   ```
2. Запустите бота:
   ```bash
   python main.py
   ```

## Основные команды (Возможен факт отсутствия функционала некоторых команд!)

### Административные команды:
- `>Set-welcome <#канал>` — Установить канал для приветственных сообщений.
- `>Del-welcome` — Удалить канал для приветственных сообщений.
- `>Set-role <@роль>` — Установить роль для новичков.
- `>Del-role` — Удалить роль для новичков.

### Команды для пользователей:
- `>Help` — Список всех доступных команд.
- `>Avatar <@пользователь>` — Получить аватарку пользователя.
- `>Hug <@пользователь>` — Обнять пользователя.
- `>Lick <@пользователь>` — Лизнуть пользователя.
- `>Slap <@пользователь>` — Шлёпнуть пользователя.

### Экономика:
- `>Card` — Показать карту с уровнем и прогрессом пользователя.
- `>Give <@пользователь> <сумма>` — Передать монеты другому пользователю.
- `>Daily` — Получить ежедневную награду.

### Информация:
- `>Serverinfo` — Информация о сервере.
- `>Userinfo <@пользователь>` — Информация о пользователе.

### Поддержка:
- `>Report <текст>` — Сообщить о баге.
- `>Feedback <текст>` — Оставить отзыв о боте.
